# sistem informasi jadwal sholat (SIFORLAT)
#### Technology
- Ruby 2.41
- Rails 5.2.1
- Devise
- Active Admin
- Ruby Slim
- Postgresql

#### Instalasi
1. Clone this repo `git clone git@github.com:muhammadyana/sistem-informasi-jadwal-sholat-siforlat.git`
1. `bundle install` or `bundle`
1. `rake db:create`
1. `rake db:migrate`
1. `rake db:seed`
1. `rails s`
1. Open browser `http://localhost:3000`
